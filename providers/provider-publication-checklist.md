# Provider Publication Checklist

- [ ] Provider entry exists in `providers/provider-ecosystem-registry.json`
- [ ] Status is valid and reviewed
- [ ] Default status is `candidate` or `tool_in_use` unless justified
- [ ] `official_partner` status includes explicit `evidence_ref`
- [ ] `logo_usage_approved` reviewed before any logo display
- [ ] Text-only card is used when `logo_usage_approved` is false
- [ ] Public copy includes required ecosystem disclaimer
- [ ] Copy does not imply endorsement, sponsorship, certification, or partnership
- [ ] No secrets, tokens, credentials, or customer data included
