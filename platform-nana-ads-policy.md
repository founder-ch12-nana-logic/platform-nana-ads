# platform-nana-ads Policy

This repository is policy-and-registry only.

## Allowed
- Planning documents
- Provider/event/campaign/budget/domain registries
- Consent requirements and approval checklists

## Forbidden
- Deployment actions
- DNS changes
- Cloudflare mutations
- Enabling Zaraz
- Live tracking scripts or pixels
- Production code
- Any ad spend execution
- Secrets or customer data storage

## Governance
All tracking and spend actions require explicit founder approval, consent readiness, and budget approval prior to any enablement outside this repository.
