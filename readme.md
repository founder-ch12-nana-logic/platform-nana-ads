# platform-nana-ads

Private ads and marketing operations repository for Nana Logic.

## Purpose
This repository stores:
- Ads planning
- Campaign registry
- Creative briefs
- Landing copy
- Channel plans
- Tracking policy
- Reporting structure

## Operational boundaries
- Do not deploy from this repository.
- Do not change DNS.
- Do not mutate Cloudflare configuration.
- Do not enable Zaraz.
- Do not add pixels.
- Do not add tracking scripts.
- Do not add secrets, tokens, passwords, or private keys.
- Do not add customer data.
- Do not auto-merge.

## Storage policy
- GitHub stores planning, registry, copy, policy, and non-secret metadata only.
- Proton Drive stores raw creative assets, videos, screenshots, and large files.
- Proton Pass stores ad account secrets, tokens, pixel secrets, and API keys.
